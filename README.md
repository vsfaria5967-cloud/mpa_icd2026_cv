# Introdução à Ciência de Dados

Este é o meu repositório da disciplina **Introdução à Ciência de Dados**, 
do [Mestrado Profissional em Administração do IFMG Campus Formiga](https://formiga.ifmg.edu.br/mestrado)

Utilizo este repositório para organizar os arquivos relacionados 
à disciplina, como dados, scripts e relatórios. Além disso, 
mantenho um diário de aprendizagem para registrar o que 
aprendi, as dificuldades que tive e o que preciso revisar.



## Como usar este projeto

Devo abrir este projeto sempre pelo arquivo `.Rproj` e manter os 
arquivos organizados nas pastas corretas.



## Estrutura básica

- `data/raw/`: dados brutos
- `data/clean/`: dados tratados
- `scripts/`: scripts `.R`
- `doc/`: relatórios e documentos em Quarto
- `listas-exercicios/`: listas de exercícios



## Registro da aprendizagem

Posso usar este arquivo para registrar, ao final de cada aula:

- o que aprendi;
- dificuldades que tive;
- o que preciso revisar.
